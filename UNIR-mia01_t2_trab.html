1. Crea una función en Python que calcule el máximo común divisor de dos números.

El MCD es el máximo número entero que divide un conjunto de dos o más números enteros sin dejar residuo.

Definimos la función mcd

In [1]:
# define la función mcd
def mcd(a, b):
    resto = 0
    while(b > 0):
        resto = b
        b = a % b
        a = resto
    return a

Solicitamos los 2 valores y calcualmos el resultado.

2. Crea una función en Python que calcule el mínimo común múltiplo de dos números.

El MCM Es el menor número natural distinto de cero que es múltiplo común de un conjunto de 2 o más números naturales

In [4]:
# define la función gcd (recursiva) y mcm
def gcd(x, y):
    return x if not y else gcd(y, x % y)
 
def mcm(x, y):
    return x * y / gcd(x, y)

def elimina_tildes(cadena):
    a,b = 'áéíóúü','aeiouu'
    trans = str.maketrans(a,b)
    return cadena.translate(trans)    
    return cadena

  4. Elimina las tildes del fichero (elimina_tildes) y convierta todas las letras a minúscula (linea_lower)

# abre fichero texto, modalidad lectura
f = open("texto.txt", "r")

# define variables de trabajo. COntador de llineas y un array "nueva_linea" para escribir en el nuevo texto.
contador_linea = 0 # inicializa en 0
ocurrencias = 0 # inicializa en 0
nueva_linea = []

# lee el texto linea a linea 
for linea in f:
        
    #contador de lineas del texto
    contador_linea = contador_linea + 1
           
    #conversor linea a minusculas. 
    linea_lower = linea.lower()
    
    #elimina tildes
    linea_sin_tildes = elimina_tildes(linea_lower)
    
    #print(linea_sin_tildes)
    
    nueva_linea.append(linea_sin_tildes)
    
    #COntador de cocurrencias de articulo "el" ya en minusculas y sin tildes
    ocurrencias = ocurrencias + linea.count("el ")
    
    
f.close()
